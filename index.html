BMI Calculator

View code

The goal of this project is to create a responsive, user-friendly web application that allows users to calculate their BMI. There are two input fields, one for weight (in kilograms) and another for height (in centimeters). A "Calculate" button that the user can click to trigger the action and a display area for the results. JavaScript is used to handle user input and calculate the BMI. The formula for calculating BMI is weight (kg) / [height (m)]^2. After the calculation, it classifies the result into the appropriate BMI category: Underweight, Normal, or Overweight.
